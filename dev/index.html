ITensorPartialDiffEq.sqeuclidean — Method

Compute the square Euclidean distance: |x - y|² = <x - y, x - y> = <x, x> - <x, y> - <y, x> + <y, y> = <x, x> - 2 * real(<y, x>) + <y, y> See: https://github.com/JuliaStats/Distances.jl

ITensorPartialDiffEq.sqeuclidean — Method

Compute the square Euclidean distance: |Ax - b|² = <Ax - b, Ax - b> = <Ax, Ax> - <Ax, b> - <b, Ax> + <b, b> = <Ax, Ax> - 2 * real(<b, Ax>) + <b, b> See: https://github.com/JuliaStats/Distances.jl
